About This Playground

This is a digital playground built by someone who knows way too much about web technologies. It's a space where I experiment with creative coding, push the boundaries of what's possible on the web, and create interactive experiences that are both technically impressive and visually stunning.

Built with pure HTML, CSS, and JavaScript - no frameworks, no libraries, just raw web technologies and a lot of creative thinking. Every experiment here is hand-crafted and optimized for performance.
